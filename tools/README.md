# Tools

This folder keeps helper files that are useful while designing or reviewing the project, but are not loaded by the production app.

## `design/`

- `design-canvas.jsx`: reusable design canvas wrapper for prototype review.
- `tweaks-panel.jsx`: reusable tweak controls for interactive prototypes.

The live app entry remains `index.html`, which currently loads `src/github-api.js` and `src/v1-magazine.jsx`.
